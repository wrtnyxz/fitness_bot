# Фитнес чат-бот (Hybrid Chatbot)

## Описание проекта
Интеллектуальная система подбора фитнес-программ тренировок в зависимости от пола и стажа пользователя. Включает автоматическую регистрацию и логирование всех действий пользователя в базу данных в реальном времени.

## Используемые технологии
- Python 3
- Django ORM & Django Admin
- pyTelegramBotAPI
- База данных SQLite

## Инструкция по установке
1. Распакуйте архив с проектом.
2. Установите необходимые библиотеки:
   pip install -r requirements.txt

## Инструкция по запуску
1. Запустите веб-сервер Django и базу данных:
   python manage.py runserver
2. В отдельном окне терминала запустите Telegram-бота:
   python manage.py run_bot
3. Перейдите в админ-панель для мониторинга логов: http://127.0.0.1:8000/admin/
